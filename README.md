# 🇮🇳 TravelBharat

A modern full-stack travel exploration platform that helps users discover famous tourist destinations across India.

TravelBharat allows users to browse destinations by state, search places, explore detailed travel information, and navigate directly using Google Maps. An integrated Admin Dashboard makes it easy to manage destinations dynamically.

---

# ✨ Features

## User Features

* Browse Indian States
* Explore Tourist Destinations
* Search Destinations
* Category-wise Filtering
* Detailed Place Information
* Google Maps Integration
* Responsive Design
* Beautiful Animations (AOS)

## Admin Features

* Add New Destination
* Update Existing Destination
* Delete Destination
* Dynamic Categories
* Real-time Database Management

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/Abhiyadav012/travelbharat.git
```

---

## Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

Run backend

```bash
npm start
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
